# Vibe Lara Kit

Laravel starter kit for fast, structured, and safe delivery.

## Stack
- Laravel
- PostgreSQL
- Docker Compose

## Quick Start
```bash
docker compose up -d --build
```

## Documentation
- Docs Portal: https://vibe-starterkit-docs.sudharmika.com

## Branching Rule
- Use one branch per topic/scope (`feat/*`, `fix/*`, `docs/*`, `chore/*`)
- Merge back to `main` after checks/review
